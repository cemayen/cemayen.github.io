Cotización

Se realiza un Levantamiento de Requerimientos de lo que necesita el cliente, y con base a lo que necesita se realiza una cotización.

Acordar Pagos

Se acordan los pagos, puede dividirse en 4, 3, 2 ó 1 pago, segun se requiera.

Aprobación de Diseño

Se realiza una propuesta de diseño con base a los requerimientos, se envia un pdf al cliente, si hay cambios en el diseño se realizan los cambios hasta que el cliente los apruebe

Maquetacion y Programación

Una vez aprovado el diseño se pasa a desarrollar el proyecto este proceso puede tardar de 1 a 5 semanas segun la complejidad del proyecto en caso de aplicaciones web se puede extender.

Entrega del Proyecto

Al terminar de programar el proyecto y que se liquide el 100% del costo, se realiza la entrega del proyecto.
